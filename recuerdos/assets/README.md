# Social preview image placeholder

To enable rich link previews (WhatsApp and other social apps) for `/recuerdos/`, manually upload a **1200 × 630 px** PNG named:

- `recuerdos-preview.png`

Place it in this folder:

- `/recuerdos/assets/recuerdos-preview.png`

This image is used by the Open Graph and Twitter metadata in `recuerdos/index.html` as the social preview thumbnail.

## Suggested design

Create a polished, screenshot-like preview of the mini app with:

- `Daniel & Flor`
- `Álbum de recuerdos`
- `Subir fotos al álbum`

Keep the style aligned with the page design (white background, gold accents, wedding/floral tone, rounded card, elegant typography).

## Important

- Do **not** add any binary image files in this PR.
